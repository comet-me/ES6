String manipulation

<script>
var name = "John Doe";
console.log(`Your name is ${name.length} characters in length`);
var msg = `${name} is a good person!`;
console.log(msg);
var description = 'this is a "good" book';
console.log(description);
description = "this is a 'good' book";
console.log(description);
description = `this is a 'good' "book"`;
console.log(description);
description = "this\tis a \\n'good book'";
console.log(description);
</script>
  • \t creates 5 spaces
  • \n create a new line
  • \\n will only print backslash n
