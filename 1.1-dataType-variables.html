Variable is a placeholder that can store a particular type of data and its value can change.

Data type tells the variable about what kind of data it can store

JavaScript is a dynamically typed language, in which data type is inferred by the JS engine while execution

<script>
var num1 = 25;
console.log(num1);
num1 = 29;
console.log(typeof (num1));
console.log(num1);
num1 = 'JavaScript is a dynamically typed language';
console.log(num1);
console.log(typeof (num1));
num1 = false;
console.log(num1);
console.log(typeof (num1));
var num2;
console.log(num2);
console.log(typeof (num2));

</script>
