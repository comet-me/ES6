Variable is a placeholder that can store a particular type of data and its value can change.

Data type tells the variable about what kind of data it can store

JavaScript is a dynamically typed language, in which data type is inferred by the JS engine while execution

<script>
var num1 = 25;
console.log(num1);
num1 = 29;
console.log(typeof (num1));
console.log(num1);
num1 = 'JavaScript is a dynamically typed language';
console.log(num1);
console.log(typeof (num1));
num1 = false;
console.log(num1);
console.log(typeof (num1));
var num2;
console.log(num2);
console.log(typeof (num2));

</script>

  • Number() converts to a Number
  • String() converts to a String,
  • Boolean() converts to a Boolean

JavaScript Data Types

5 different data types that can contain values:

  1. string
  2. number
  3. boolean
  4. object
  5. function

There are 3 types of Objects

  1. Oject
  2. Date
  3. Array

And 2 data types that cannot contain values:

  1. null
  2. undefined

Note: You cannot use typeof to determine if a JavaScript object is an array (or a date).
