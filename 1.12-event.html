Events triggered:

  • An HTML web page has finished loading
  • An HTML input field was changed
  • An HTML button was clicked

List HTML common events:

HTML events are "things" that happen to HTML elements; can be somethings the browser does, or something a user does.

  • onchange An HTML element has been changed
  • onclick The user clicks an HTML element
  • onmouseover The user moves the mouse over an HTML element
  • onmouseout The user moves the mouse away from an HTML element
  • onkeydown The user pushes a keyboard key
  • onload The browser has finished loading the page

What can JavaScript Do?

Event handlers can be used to handle, and verify, user input, user actions, and browser actions:

  1. Things that should be done every time a page loads
  2. Things that should be done when the page is closed
  3. Action that should be performed when a user click a button
  4. Content that should be verified when a user inputs data, And more...
  • HTML event attributes can execute JavaScript code directly
  • HTML event attributes can call JavaScript functions
  • You can assign your own event handler functions to HTML elements
  • You can prevent events from being sent or being handled, And more...
